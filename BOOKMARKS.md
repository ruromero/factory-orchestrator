# BOOKMARKS.md

References for agents to load on demand.

## Go

- [net/http](https://pkg.go.dev/net/http) — HTTP client/server. Used for all API clients.
- [encoding/json](https://pkg.go.dev/encoding/json) — JSON marshaling/unmarshaling.
- [log/slog](https://pkg.go.dev/log/slog) — Structured logging (Go 1.21+).

## APIs

- [Ollama API](https://github.com/ollama/ollama/blob/main/docs/api.md) — Chat, model management, tool calling.
- [Gemini API](https://ai.google.dev/gemini-api/docs) — Content generation, research.
- [GitHub REST API](https://docs.github.com/en/rest) — Issues, PRs, comments, labels.

## MCP

- [MCP Specification](https://modelcontextprotocol.io/specification) — JSON-RPC protocol for tool servers.
- [Serena](https://github.com/oraios/serena) — LSP-based MCP server for code navigation.
- [Context7](https://github.com/upstash/context7) — Library documentation MCP server.

## Design patterns

- [fullsend](https://github.com/fullsend-ai/fullsend) — Design patterns for autonomous agentic development.
